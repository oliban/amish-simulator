Characters Directory
====================

Purpose: Player character and NPC meshes/sprites.

Assets needed:
- character_base: Main character mesh (male/female Amish variants)
- beard_mesh: Procedural beard geometry for male characters
- Bonnet variants for female characters
- Plain clothing textures (suspenders, plain dress)

Kenney packs to reference:
- Tiny Dungeon (16x16 character sprites as placeholders)
- Nature Kit does NOT include character models

License: All Kenney assets are CC0. Custom character art must be original.

Notes:
- Characters need to support the BeardSystem (progressive beard growth)
- AgingSystem requires age-based appearance changes
- Consider using Unity's sprite renderer for 2D or simple 3D capsules as placeholders
