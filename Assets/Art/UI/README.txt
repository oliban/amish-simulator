UI Directory
============

Purpose: HUD icons, menu elements, and interface graphics.

Assets needed:
- icon_energy: Energy/stamina bar icon
- icon_hunger: Hunger meter icon
- icon_time: Time-of-day indicator icon
- character_male: Male character portrait/icon
- character_female: Female character portrait/icon
- Menu backgrounds and button styles

Kenney packs to reference:
- Game Icons: gear.png, home.png, power.png, star.png, arrows, audio controls
  (Located at Art/kenney/kenney_game-icons/PNG/White/1x/)
- Game Icons Vector: SVG versions for scaling
  (Located at Art/kenney/kenney_game-icons/Vector/)

License: All Kenney assets are CC0.

Notes:
- Game Icons pack provides generic UI controls (arrows, buttons, gear, home)
- Energy/hunger/time icons need to be custom-made or adapted from existing icons
- Consider tinting white PNG icons with Unity's Image.color for theming
- The Pixel Platformer Farm Expansion may have useful crop/farm icons
