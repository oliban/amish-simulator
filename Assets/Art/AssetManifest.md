## Custom Art Required

The following assets were NOT found in any CC0 pack and will need to be created:

| Asset | Description | Priority | Notes |
|-------|-------------|----------|-------|
| Butter churn model | 3D prop for the butter churning mini-game | High | Not found on poly.pizza, OpenGameArt, or Kenney |
| Plow model | 3D prop for the plowing mini-game | High | Not found; could be kitbashed from Nature Kit + Fantasy Town Kit parts |
| Amish clothing materials | Suspenders, bonnet, plain dress textures for character re-skin | High | Apply to animated_characters BaseCharacter mesh |
| Bonnet mesh | Female head accessory | Medium | Simple geometry, ProBuilder-buildable |
| Beard mesh/shader | Procedural beard growth | High | Custom shader needed |
| Seasonal crop materials | Color/texture variants | Medium | Nature Kit + crops pack provide base meshes |
| UI HUD icons | Energy/hunger bars, time-of-day | Medium | Game Icons pack provides base |
| Quilt patterns | Texture patterns for quilting mini-game | Medium | Texture work only |

### Notes on Available Assets for Custom Work
- **Barn:** Barn.glb, Big_Barn.glb, Small_Barn.glb, Open_Barn.glb are all available and CC0
- **Horse:** Horse.glb (animated, GLB format) ready to use
- **Wagon/Cart:** Cart.glb and Broken_Cart.glb available CC0; WoodWheel.glb for detail
- **Barrel:** Barrel.glb (CC0, from poly.pizza)
- **Hay bale:** Hay.glb (CC0, from poly.pizza)
- **Fence post:** Fence_1.glb and Fence_2.glb (CC0, from farm buildings bundle); also in platformer-kit and fantasy-town-kit
- **Human characters:** BaseCharacter.fbx (animated, CC0) in animated_characters pack; Worker_Male/Female closest to Amish
