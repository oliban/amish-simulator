- kenney_tiny-town/        - 130 pixel tiles (16x16, town buildings)
- kenney_tiny-dungeon/     - 130 pixel tiles (16x16, characters/items)
- kenney_pixel-platformer-farm-expansion/ - 110 pixel tiles (18x18, farm theme)
- kenney_isometric-roads/  - 96 isometric road tiles (PNG)
